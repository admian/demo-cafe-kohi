# Kōhī House — Cafe Landing Page

A warm, elegant cafe landing page demo built with **Vue 3**, **TypeScript**, **Vite**, and **Tailwind CSS**.

![Vue](https://img.shields.io/badge/Vue-3-4FC08D?logo=vue.js)
![TypeScript](https://img.shields.io/badge/TypeScript-5-3178C6?logo=typescript)
![Tailwind CSS](https://img.shields.io/badge/Tailwind-4-06B6D4?logo=tailwindcss)

## Live Demo

https://demo-cafe-kohi.vercel.app

## Features

- Responsive design with serif + sans typography
- Hero, menu grid, about section with image carousel, visit & reservation form
- Warm earthy color palette
- Smooth scroll navigation
- Interactive buttons with toast feedback

## Tech Stack

- Vue 3 + TypeScript (Composition API / `<script setup>`)
- Vite 7
- Tailwind CSS 4

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

```bash
npm run build
npm run preview
```
